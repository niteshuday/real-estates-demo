
# Premium Real Estates — Demo Website

Professional, responsive showcase for curated luxury property listings. This lightweight static site demonstrates a production-ready frontend built with Vite, Tailwind CSS, and modern JavaScript — optimized for fast load times, crisp imagery, and an accessible, mobile-first layout.

**Live demo**: https://niteshuday.github.io/real-estates-demo/

Why this project
-----------------
- **Performance-focused**: built with Vite for fast dev experience and highly optimized production bundles.
- **Design-ready**: responsive grid and image handling for retina and mobile devices.
- **Data-driven**: property data is simple JSON in `propertiesData.js` so you can extend or integrate a backend easily.
- **i18n-ready**: sample translations in `translations.js` show how to localize strings.

Highlights
----------
- Fast local development: `npm run dev`
- Production build: `npm run build` (outputs to `dist/`)
- GitHub Pages deploy: `npm run deploy` (uses `gh-pages`)

Quick start
-----------
1. Install dependencies

```bash
npm install
```

2. Run locally (development server)

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

4. Deploy to GitHub Pages

```bash
npm run deploy
```

Project structure (important files)
----------------------------------
- `index.html` — main HTML entry
- `src/main.js` — app entrypoint
- `src/propertiesData.js` — example property data
- `src/translations.js` — sample localization strings
- `vite.config.js` — build configuration
